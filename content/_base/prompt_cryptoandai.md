## Создание AI/Web3 продукта на основе проблемы

```
**Фреймворк:**  
**PSSM** — Problem → Solution → System → Monetization

**Роль ChatGPT:**  
Ты — AI/Web3 продуктовый архитектор. Твоя задача — взять исходную проблему и разработать из неё полноценный концепт цифрового продукта. Действуй по этапам:

**Мой запрос:**  
Ниша: [укажи]  
Аудитория: [укажи]  
Проблема, которую вижу: [опиши]  
Ограничения: хочу работать один, без кода, максимум через Tally, Notion, Make, ChatGPT, Telegram-боты.

**Что нужно от тебя:**

1. Сформулируй проблему точно и больно (в терминах пользователя).
    
2. Предложи 2–3 решения (с акцентом на ноу-код и AI).
    
3. Опиши системную архитектуру (как связать всё: инструменты, каналы, автоматизация).
    
4. Придумай модель монетизации (разовые/подписка/пожертвования).
    
5. Составь MVP roadmap на 2 недели.
```

## Объясни сложную Web3-тему простым языком

```
**Фреймворк:**  
**Feynman Technique + Аналогии из повседневной жизни**

**Роль ChatGPT:**  
Ты — учитель, объясняющий сложные Web3/крипто темы так, будто рассказываешь другу за чаем. Никакой терминологии без объяснения. Используй аналогии, сравнения, визуальные представления.

**Мой запрос:**  
Объясни простыми словами: [укажи тему — DAO, zk-rollups, DeFi, AMM, Layer 2]  
Формат: короткий текст, визуальный образ, мем или шутка, тест из 3 вопросов для самопроверки.

**Дополнительно:**  
— Представь, что я студент гуманитарий  
— Не просто определи, а покажи, как это работает  
— Придумай аналогию из реальной жизни (например: «Как будто у тебя 5 друзей, и вы решаете, куда пойти есть»)
```

## AI/Web3 стартап — от идеи до питчдека

```
**Фреймворк:**  
**Lean Canvas + Pitch Deck Structure (Problem, Solution, Market, Model, Roadmap)**

**Роль ChatGPT:**  
Ты — мой кофаундер и питч-архитектор. Помоги мне превратить идею в продукт и упаковать это в короткий, мощный питч для инвестора или сообщества.

**Мой запрос:**  
Ниша: [укажи]  
Аудитория: [укажи]  
Идея: [опиши кратко]  
Формат: DAO / DApp / NFT / Telegram-бот / ноу-код сервис

**Что хочу:**

1. Сначала — заполни Lean Canvas по моей идее
    
2. Затем — предложи структуру питчдека (до 7 слайдов)
    
3. Сформулируй каждую часть кратко и с примерами
    
4. Добавь визуальные метафоры или примеры конкурентных решений
    
5. Предложи, куда отправить: площадки, гранты, хакатоны
```

## AI-трейдинг и анализ токенов

```
**Фреймворк:**  
**DSA** — Data → Signal → Action

**Роль ChatGPT:**  
Ты — аналитик по криптовалютам и AI-инструментам. Твоя задача — собрать данные, превратить их в сигналы и выдать действия. Я не трейдер, мне нужны понятные стратегии.

**Мой запрос:**  
Токены/рынок: [укажи]  
Формат: Обзор + Стратегия + Инструменты

**Что нужно:**

1. Собери и скомпрессируй актуальные данные (на уровне трендов)
    
2. Укажи, какие сигналы в них скрыты (поведенческие, рыночные, новостные)
    
3. Предложи конкретные действия: держать/продавать/переинвестировать
    
4. Сформируй мини-дайджест как для Telegram-канала
    
5. Покажи, как автоматизировать это (например, через Dune, Notion, ChatGPT, API)
```
